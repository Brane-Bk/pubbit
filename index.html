Guide d'installation pas-à-pas — PubBit

PubBit est une extension légère qui vous récompense en Sats pendant votre navigation via le réseau Lightning. Disponible pour Chrome et Firefox. Suivez ces 4 étapes simples ; chaque étape contient une grande image pour vous guider visuellement.

1 — Télécharger & dézipper

Téléchargez le fichier depuis pubbit.com. Le fichier est généralement nommé pubbit-x.y.z.zip. Une fois le téléchargement terminé, dézippez-le sur votre ordinateur (clic droit → Extraire / unzip).

Vous obtiendrez un dossier contenant au minimum : manifest.json, un dossier icons/ et un dossier src/. Conservez ce dossier accessible (par exemple sur le Bureau) pour l'étape d'installation.

2 — Ouvrir Chrome ou Firefox

Choisissez le navigateur dans lequel vous souhaitez installer PubBit. Ces instructions couvrent Chrome (Chromium, Edge basé sur Chromium) et Firefox. Si vous utilisez un autre navigateur, suivez la procédure correspondante sur notre site.

Important : pour Chrome, assurez-vous d'avoir une version récente et d'activer le mode développeur lors de l'installation locale. Pour Firefox, on utilisera la page about:debugging pour charger temporairement l'extension.

3 — Mode développeur (Chrome) ou page de debug (Firefox)

Chrome / Chromium

  1. Ouvrez l'adresse chrome://extensions/ dans la barre d'adresse.
  2. Activez le Mode développeur (coin supérieur droit).
  3. Cliquez sur Charger l'extension non empaquetée.
  4. Sélectionnez le dossier dézippé (le dossier qui contient manifest.json).
  5. L'extension apparaîtra dans la liste et s'activera immédiatement. Vous pouvez épingler l'icône si vous le souhaitez.

Firefox

  1. Ouvrez about:debugging#/runtime/this-firefox dans la barre d'adresse.
  2. Cliquez sur Charger un module complémentaire temporaire.
  3. Sélectionnez le fichier manifest.json dans le dossier dézippé.
  4. L'extension sera chargée pour la session courante (temporaire). Pour une installation permanente, suivez la procédure de publication sur AMO ou utilisez un XPI signé.

Ces étapes permettent d'installer localement en développement. Si vous préférez, installez PubBit directement depuis le Chrome Web Store ou AMO (Firefox) si la version y est disponible.

4 — Charger et activer

Après avoir chargé l'extension non empaquetée, vérifiez les permissions demandées (accès aux onglets, permissions de stockage, etc.). Activez l'extension et cliquez sur l'icône pour ouvrir le panneau PubBit.

Après chargement temporaire, cliquez sur l'icône dans la barre d'outils pour ouvrir PubBit. Notez que le chargement temporaire disparaît après redémarrage ; pour usage prolongé, installez depuis AMO.

Une fois installée, PubBit peut vous demander d'autoriser une connexion à votre portefeuille Lightning. Ceci est nécessaire pour créditer vos Sats. Les paramètres de l'extension permettent d'ajuster la fréquence et les préférences de monétisation.

Description complète de PubBit

PubBit est une extension qui rémunère votre attention en distribuant des fractions de Bitcoin (Sats) via Lightning. Elle fonctionne en arrière-plan, évalue anonymement l'engagement et propose des micropaiements rémunérateurs. Caractéristiques :

  • Confidentialité : pas de tracking centralisé, les données restent locales.
  • Rapide : paiements Lightning instantanés.
  • Contrôlable : paramètres pour limiter la fréquence et les types d'annonces.
  • Portable : disponible sur Chrome & Firefox.

Pour toute question ou pour signaler un problème, rendez-vous sur pubbit.com ou contactez-nous via la page Contact.
